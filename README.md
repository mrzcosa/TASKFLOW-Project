# Cosa ToDo Dashboard

A full-stack task management system built using:

- ASP.NET Core Minimal API
- Entity Framework Core
- MySQL
- HTML
- CSS
- JavaScript

## Features

- Create Tasks
- Delete Tasks
- Search Tasks
- Due Date Tracking
- Priority Management
- Dashboard Statistics

## Tech Stack

- C#
- ASP.NET Core
- MySQL
- Entity Framework Core
- HTML5
- CSS3
- JavaScript
